# 📒 진행 방식
- **매주 일요일 22:00** 스터디를 진행합니다.
- **토요일 자정 12시**까지 '발표가 가능할 수준'까지 학습하며 학습 자료 PR 남깁니다.
- 스터디 당일, 모든 팀원이 돌아가며 학습 내용 발표합니다.
    - 이해가 잘 되지 않는 부분이나 질문이 있다면 함께 피드백
- 발표가 모두 종료되었다면 다음 스터디에서의 학습할 양과 담당자 정하고 스터디를 마칩니다.
